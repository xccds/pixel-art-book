# 附录A 资源罗盘（像素向）

> 按"我需要…"的场景索引，不是按类型罗列。每个条目都是像素创作中用得上的——收录标准很简单：我自己用过、或者像素社区广泛验证过。没有广告，没有 affiliate link。

---

### A.1 我需要找颜色 / 色板 / 色彩参考

**[Coolors](https://coolors.co)** — 按空格键随机生成色板，锁定喜欢的颜色继续迭代，支持导出到 Aseprite。

**[Adobe Color](https://color.adobe.com)** — 基于色彩理论（补色/类比/三色）的调色工具，可以从图片提取色板。

**[Lospec Palette List](https://lospec.com/palette-list)** — 像素艺术专用的色板数据库，每套色板标明颜色数量和适用场景。你如果不知道"我的游戏该用几色"，先在这里翻一小时。

**[Pico-8 Palette](https://lospec.com/palette-list/pico-8)** — 游戏业界最有名的 16 色调色板，低分辨率游戏的经典之选。

**[Palettte](https://palettte.app)** — 色板微调工具，可以在渐变图上拖拽调节单个颜色，看看它在色板全局中的位置。做 palette ramp 时的必备神器。

---

### A.2 我需要像素艺术专用工具 / 教程 / 色板

**[Lospec](https://lospec.com)** — 像素艺术的"一站式入口"：色板、教程、工具列表、艺术家索引。如果你只能收藏一个像素艺术网站，就这个。

**[Aseprite](https://www.aseprite.org)** — 像素艺术动画软件的事实标准。支持洋葱皮、精灵表导出、自定义色板和调色旋转。收费（$19.99），但完全值回票价。

**[LibreSprite](https://libresprite.github.io)** — Aseprite 的开源分支，如果你预算为零，这是最佳替代。

**[Pyxel Edit](https://pyxeledit.com)** — 专注于 tileset 编辑的像素工具，自动拼接预览对做 tile 地图非常方便。

**[Tiled](https://www.mapeditor.org)** — 免费开源的 tile 地图编辑器，支持等距和六边形地图，直接导出 TMX 格式给 Godot/Unity。

**[Piskel](https://www.piskelapp.com)** — 在线像素编辑器，不需要安装。适合"我还没决定要不要买 Aseprite，先试试画像素是什么感觉"的阶段。

**[Lospec Blender Toolkit](https://lospec.com/blender-toolkit/)** — 免费、单文件即用的 3D→像素渲染工具包。如果你要做制作06 的"3D 当 IR"管线，从这里起步最快。

---

### A.3 我需要找游戏美术参考 / UI 参考

**[Mobbin](https://mobbin.com)** — 真实 App UI 截图的数据库，按页面类型搜索。做游戏 HUD 和菜单时的跨领域参考——手游 UI 的设计质量通常高于独立游戏 UI。

**[Game UI Database](https://www.gameuidatabase.com)** — 专门收录游戏内 UI 截图的数据库，按类型/风格/元素分类。做制作04 的 HUD 练习前，先在这里搜"pixel art inventory"看 20 张截图。

**[Interface In Game](https://interfaceingame.com)** — 按游戏名索引的 UI 截图合集，每个游戏拆解了 HUD、菜单、对话框等区域。

**[Dribbble](https://dribbble.com)** — 用 "game UI" "pixel art HUD" "inventory screen" 等关键词搜索。注意区分概念稿和实装稿——概念稿不需要真的能跑。

---

### A.4 我需要 2D 像素游戏美术风格分类的直观参考

**[Spriters Resource](https://www.spriters.net)** — 从商业游戏中提取的精灵图和 tileset 资源库。用于研究"某个游戏的精灵实际长什么样"——像素尺寸、帧数、色板。不是用来复制，是用来做技术拆解。

**[Pixel Joint](https://pixeljoint.com)** — 像素艺术社区画廊，按主题/风格/色板浏览。做某种风格前，先搜类似的作品看看别人怎么处理那个像素尺寸下的体积、明度、轮廓。

**[The Spriters Resource (GBA/NDS 区)](https://www.spriters-resource.com)** — 特别推荐 GBA 和 NDS 分区——这些是"低分辨率像素美术的黄金时代"，每一个精灵都是在极限约束下做的。用观察03 的 VTS 法拆解一张 GBA 游戏的精灵表，比看 10 个教程有用。

---

### A.5 我需要字体 / 排版参考

**[Google Fonts](https://fonts.google.com)** — 免费字体库，按分类筛选，支持预览你要的文本在字体下的效果。

**[DaFont](https://www.dafont.com)** — 按风格（像素/手写/哥特/Sci-Fi）索引的字体站。注意看授权协议——大部分只能免费用于个人项目。

**[Font Squirrel](https://www.fontsquirrel.com)** — 100% 可商用的免费字体精选集，量不大但每个都经过授权验证。

---

### A.6 我需要像素艺术社区 / 反馈 / 日常看图

**[Pixel Joint](https://pixeljoint.com)** — 像素艺术最老牌的社区画廊。按色板/主题/风格浏览，每周有挑战赛。适合"我想看看别人用 16 色能画出什么"的阶段。

**[Reddit r/PixelArt](https://www.reddit.com/r/PixelArt/)** — 像素艺术 Reddit 社区，日常更新量大，适合每天翻 5 分钟喂眼睛。

**[Twitter/X #pixelart](https://twitter.com/search?q=%23pixelart)** — 像素艺术日常创作的最大聚集地。关注几个你喜欢的像素艺术家，每天刷到即"被动喂眼睛"。

**[DeviantArt Pixel Art](https://www.deviantart.com/tag/pixelart)** — 体量极大，质量方差也极大。用 "pixel art landscape" "pixel art character" 精确搜索。注意——这里不是看"最好"的，是看"各种水平"的，对训练你的判断力有帮助。

**[Lospec Discord](https://lospec.com/discord)** — Lospec 的 Discord 社区，可以贴作品求反馈。像素社区整体友善、反馈具体——不像某些艺术社区只说"好看"。

---

### A.7 我需要提升日常"看"的能力

**[ArtStation](https://www.artstation.com)** — 游戏/影视概念设计领域的主流作品平台。用这里找"顶级标准长什么样"。缺点是容易焦虑——记住，这些是行业顶尖，不是你 3 个月的对照标准。

**[Pinterest](https://www.pinterest.com)** — 用它建你自己的参考板。每看到一个你觉得"这个可以用"的画面，钉进去。30 天后你会有一个个人审美参考库。技巧：不要只钉"好看"的，也要钉"不好看但我知道为什么不好看"的——后者更有教学意义。

**[PureRef](https://www.pureref.com)** — 参考图管理工具。把参考图拖进去自由排列、缩放、旋转，存成一个 .pur 文件。做风格文档时的必备——把 Gold Standard 图和所有参考图放一屏，一眼看出"偏了没有"。

---

### A.8 我需要游戏原型 / 角色生成 / 占位素材

**[Kenney](https://kenney.nl)** — 游戏原型素材的"行业金标准"。从 2D 精灵到 UI 元素到音效，全部 CC0，风格统一，可以直接用在 Game Jam 成品中。

**[OpenGameArt](https://opengameart.org)** — 开源游戏素材社区，按授权协议筛选。量很大但质量方差也大——需要挑。

**[Itch.io Assets](https://itch.io/game-assets)** — Itch.io 上的游戏素材市场，有大量免费/付费选项。很多独立像素艺术家的色板和精灵包在这里首发。

**[SpriteLib](https://www.spritelib.com)** — 免费 2D 精灵资源，适合做占位素材——先用占位跑通玩法，再自己画替换。

---

### A.9 我需要了解各种美学风格的定义和案例

**[Aesthetics Wiki](https://aesthetics.fandom.com/wiki/Aesthetics_Wiki)** — 一个记录"所有互联网美学运动"的维基。从 Vaporwave 到 Cottagecore 到 Frutiger Aero，每个风格有定义、核心色板、视觉元素清单和大量图例。对游戏风格选择来说是一座尚未被充分开采的金矿。

**[CARI](https://cari.institute)** — 消费者审美研究所，按年代和风格归档了"主流消费品的视觉语言"。适合理解"某种风格为什么在那个年代流行"。

---

### A.10 我需要分析电影构图 / 镜头语言

**[FILMGRAB](https://film-grab.com)** — 电影逐帧截图库，按导演/电影名索引，适合研究构图、打光、色调。

**[Movies In Color](https://moviesincolor.com)** — 把电影帧分解成色谱，直观展示一部电影的调色板结构。看完你会意识到——所有好电影都有明确的调色板约束，和你的游戏一样。

---

### A.11 我需要在线上或者离线学画画基础

**[Drawabox](https://drawabox.com)** — 免费的、从零开始的结构化线稿课程。以"每天画一个盒子"为基础，建立空间感和线控能力。不适合零耐心的人——需要跟完作业。

**[Ctrl+Paint](https://www.ctrlpaint.com)** — 免费的数字绘画入门教程，按"基础技能"和"实际项目"分两条学习路径。视频短小（大多 5 分钟），很适合碎片时间学习。

**[Proko](https://www.proko.com)** — 人体解剖和肖像绘画的标杆教程，大部分免费。做角色设计时理解头颈肩结构的绝佳入口。

---

> **使用建议：** Lospec、Kenney、Game UI Database、Pixel Joint、PureRef——这五个值得加入书签栏。其余的按你现在卡在哪一章决定看哪些。做练手05 卡色彩，先翻 A.1；做制作03 卡 tile，先翻 A.2 看 Tiled 和 Pyxel Edit；做制作04 卡 HUD，先翻 A.3 看 Game UI Database。
