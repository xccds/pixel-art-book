# 附录C 练习速查卡

> 全书练习的表格缩略版。右列"核心步骤缩略"是为已经做过一遍、回来快速回忆用的——第一次练请回原文看完整说明。
>
> **使用建议：** 不是练完所有 56 项就通关。挑你当前章节的 3-5 个练习先做，做完了再回来选下一组。L3 练习建议在周末或 Game Jam 中安排整块时间。
>
> **难度定义：** L1 = 30 分钟内可完成，不依赖任何前置技能。L2 = 需要 1-4 小时，依赖基础概念。L3 = 需要半天以上或跨章整合。
