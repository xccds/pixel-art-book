# 附录D 自评量表

> 基于观察01 的三能力自评模型，升级为月度追踪版本。每月最后一天花 10 分钟填一次，三个月后看雷达图——进步的形状比分数本身更诚实。

---

## D.1 三能力月度自评表

**评分标准（1-5 分）：**

- **1 分** — 完全不知道这是什么，也看不出好坏
- **2 分** — 知道概念，但自己看不出来，更做不出来
- **3 分** — 能识别别人的问题（"这里 AA 没做""这里 banding 很明显"），但自己做也犯同样的错
- **4 分** — 大多数时候能做出对的决策，少数需要查文档/对照参考
- **5 分** — 不需要对照参考，直觉就能做出对的决策，且能向别人解释为什么对

| 能力维度 | 含义（一句话） | 第1月 | 第2月 | 第3月 |
|----------|---------------|-------|-------|-------|
| **看** `Kan` | 你能"看"到什么——能否注意到明度对比、构图失衡、色彩不和谐、jaggies/banding、质感不一致 | ___/5 | ___/5 | ___/5 |
| **做** `Zuo` | 你能否"画"出来——手里有没有肌肉记忆、能否稳定复现你心里想要的画面、工具快捷键不卡壳 | ___/5 | ___/5 | ___/5 |
| **整** `Zheng` | 你能否把看和做串成完整管线——从风格决策到画出来到导出 PNG 到导入引擎到一致性审计 | ___/5 | ___/5 | ___/5 |

---

## D.2 如何使用本量表

本量表的设计目的不是"打分排名"，而是**把隐性的进步变成可见的形状**。你在第 1 个月可能各项都是 2 分——这是正常的。三个月后再看，如果某个维度从 2 变成了 4，你知道"这个方法有效"。如果某个维度连续两个月停在 2，你知道"这个攻法可能不对，要换一种"。

**填写节奏：**

- 每月最后一天，固定场所（比如睡前书桌前），固定工具（同一支笔或同一个 Google Sheets）
- 填分数前，先翻看本月的产出截图——让你的分数基于证据，而不是基于"感觉我今天画得不错"
- 填完分数后立即填 D.4 的反思三问——不要等到第二天，那时的记忆已经被你的大脑"重写"过了
- 三能力的含义不是孤立的：**看**好才能**做**好，**做**好才能**整**好。如果你的"整"卡在 2 但"做"是 4——你的问题是管线，不是画技

## D.3 雷达图绘制说明

你可以在纸上手绘，也可以用 Google Sheets / Excel / Numbers 自动生成。以下两种方法任选。

### 方法一：纸上手绘（推荐第一个月）

1. 在纸上画一个正三角形，从中心向外画 3 根轴线，分别标注"看/做/整"
2. 每条轴线从中心到边缘分为 5 等份（1, 2, 3, 4, 5）
3. 每月最后一天，在三条轴上标上本月分数，用直尺连线
4. 三个月后你面前有三条线（第 1 月实线、第 2 月长虚线、第 3 月短虚线）——三角形面积变大 = 整体进步；某个顶点突出 = 强项明确；趋于正三角形 = 均衡发展

### 方法二：Google Sheets / Excel 自动生成

1. 新建 Sheet，A 列放三个维度名（看/做/整），B-D 列放三个月分数
2. 选中数据 → 插入图表 → 选择"雷达图"（有些版本叫"蛛网图"或"极坐标面积图"）
3. 添加三个数据系列（每月一条），用不同颜色区分
4. 第 1 月实线，第 2 月虚线，第 3 月点线

## D.4 月度自评反思三问

填完分数后，回答以下三行（这三行比分数本身更有用）：

| 问题 | 第1月回答 | 第2月回答 | 第3月回答 |
|------|----------|----------|----------|
| **本月哪个维度进步最大？** 为什么？（具体事件：做了哪个练习、画了哪张图、看了哪本书） | | | |
| **本月哪个维度原地踏步？** 卡在哪？（具体卡点：哪个概念不懂、哪个工具不熟、哪类画不对） | | | |
| **下个月只攻一个维度——攻哪个？** 为什么选它？打算用哪 3 个练习攻？（从附录C 挑） | | | |

---

## D.5 季度回顾清单（每三个月做一次）

- [ ] 把三个月雷达图叠在一起——三角形面积是变大了还是只是形状变了？面积变大说明各项都在进步；形状趋于正三角说明均衡发展；某个顶点拉得很长但另外两个缩着——你可能在"偏科"
- [ ] 找出对自己最有效的练习类型：是临摹进步大，还是自己从零画进步大？是跟着教程一步步走有效，还是先自己试错再看教程有效？把这个结论记下来——它决定了你下个季度的时间分配
- [ ] 翻出三个月前画的第一张画和最近一张——并排放着看 30 秒，写下三处具体的不同（不只是"更好看了"，而是"线条更干净了""明度对比更清晰了""色彩饱和度控制更稳定了"）
- [ ] 更新你的"当前卡点"诊断：三个月前的卡点还在吗？如果还在，你用的攻法是不是需要换？新卡点是什么？把它写进下一张月度自评表的"下月攻哪个维度"那一行
- [ ] 如果某维度连续两个月没动，回观察01 重看那一层的诊断部分。换个方法攻——别用同样的练习再攻一次然后期待不同结果
- [ ] 对照附录C 的练习速查卡——你已经做过的练习中，哪些在"看"列、哪些在"做"列、哪些在"整"列？如果三个月下来全部 L1 都在"做"列——你可能"画了很多但看得不够"，该补观察部的练习了

---

> **一句话心法：** 自评不是考试——不是分数越高越好，是"知道自己在哪"本身比"在哪"更重要。一个诚实的 2 分比一个虚假的 4 分有用十倍。
