# 附录G Aseprite 快捷键

> 按类别整理的 Aseprite 最常用快捷键速查表。不需要全背——画一个月后肌肉自然会记住。如果你是第一次打开 Aseprite，先看制作01 的入门引导。

---

## G.1 工具（画什么用哪个键）

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **B** | 铅笔 (Pencil) | 像素艺术的核心工具——一格一个像素，无抗锯齿。你 90% 的时间在这个工具上 |
| **E** | 橡皮擦 (Eraser) | 擦除像素——等价于用背景色画。右键 = 快速切换橡皮擦 |
| **G** | 油漆桶 (Paint Bucket) | 填充连续同色区域。**注意：** 确保 Tolerance=0，否则会漏到相邻色 |
| **I** | 取色器 (Eyedropper) | 按住 Alt 在你画的地方取色——比去色板窗口点快 10 倍 |
| **L** | 直线工具 (Line) | 画直线——练手01 的"无锯齿曲线"练习的主力工具 |
| **Shift+I** | 矩形选框 | 选区工具——选一个矩形区域后可以移动/复制/变换 |
| **U** | 矩形工具 (Rectangle) | 画空心/实心矩形——按住 Shift 画正方形 |
| **M** | 选区工具 (Marquee) | 矩形选区——和 Shift+I 功能类似，快捷键不同 |
| **W** | 魔棒 (Magic Wand) | 选中所有连续同色像素——用于"我要把整片天空换一种颜色" |
| **H** | 手型 (Hand) | 拖动画布——但更推荐：按住空格键拖动（不需要切到 H 工具） |
| **Z** | 缩放 (Zoom) | 点击放大——但更推荐：Ctrl+滚轮 或 数字键 1-6 |
| **S** | 涂抹 (Blur) | 像素艺术几乎不用——注意不要误触 |
| **T** | 文字 (Text) | 输入文字——做 UI 和标题时用 |
| **R** | 旋转 (Rotate) | 旋转选区或图层——像素艺术中旋转会产生大量 jaggies，需要手动修 |

---

## G.2 图层（组织你的画面）

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Shift+N** | 新建图层 | 弹窗确认图层名和属性 |
| **Alt+N** | 在当前图层上方新建 | 快速加一层，不弹窗 |
| **Ctrl+Shift+N** | 新建图层组 | 把相关图层装进文件夹——"背景层组""角色层组""UI 层组" |
| **F7** | 显示/隐藏图层面板 | 图层管理的主面板切换 |
| **Ctrl+Shift+[** | 当前图层下移 | 调整图层叠加顺序 |
| **Ctrl+Shift+]** | 当前图层上移 | 调整图层叠加顺序 |

---

## G.3 帧/动画（让精灵动起来）

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Alt+N** | 新建帧 | 在当前帧后面加一帧空白帧（和图层 Alt+N 相同，取决于焦点在哪） |
| **Alt+Left** | 上一帧 | 回到前一帧 |
| **Alt+Right** | 下一帧 | 前进到后一帧 |
| **Shift+Left** | 跳到第一帧 | 返回动画开头 |
| **Shift+Right** | 跳到最后一帧 | 跳到动画末尾 |
| **Enter** | 播放/停止动画 | 预览你的逐帧动画——制作05 最常用的键 |
| **F7** | 显示/隐藏时间轴 | 切换时间轴面板（帧列表+图层列表在一个面板） |
| **Ctrl+D** | 复制当前帧 | 在逐帧动画中复用前一帧作为基础、只改变化的部分 |

---

## G.4 视图（怎么看你的像素）

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **1** | 缩放到 100% | 像素艺术的"真实尺寸"——这才是玩家看到的 |
| **2** | 缩放到 200% | 最常见的编辑视图 |
| **3** | 缩放到 300% | 中距离编辑 |
| **4** | 缩放到 400% | 细节编辑 |
| **5** | 缩放到 600% | 极度放大——画 16×16 精灵时用 |
| **6** | 缩放到 800% | 最大缩放——用于检查单个像素的 AA 和 banding |
| **+ / -** | 放大/缩小 | 逐级缩放 |
| **Ctrl+0** | 适配屏幕 | 让画布充满整个编辑区 |
| **Space + 拖动** | 平移画布 | 比切到 H（手型工具）快——按住空格不放拖 |
| **Ctrl+`** | 显示/隐藏网格 | 像素网格线——画 tile 和检查对齐时必开 |
| **Ctrl+Shift+`** | 显示/隐藏像素网格 | 更细的像素格——1x 缩放下的格子 |
| **Tab** | 切换全屏模式 | 隐藏所有面板，只看画布——做"3 秒扫视测试"（制作09 L2）时用 |

## G.5 文件（存盘不能忘）

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Ctrl+N** | 新建文件 | 设置画布尺寸——32×32 是你的默认值 |
| **Ctrl+O** | 打开文件 | 打开 .ase/.aseprite/.png 文件 |
| **Ctrl+S** | 保存 | 保存为 .aseprite（保留图层和帧） |
| **Ctrl+Shift+S** | 另存为 | 另存为新文件——第一次导出 PNG 时用"另存为"选 PNG 格式 |
| **Ctrl+Alt+Shift+S** | 导出精灵表 | 把多层多帧的 .aseprite 导出为单张精灵表 PNG——制作02 的关键步骤 |
| **Ctrl+W** | 关闭当前标签 | 关闭当前打开的文件标签页 |

---

## G.6 编辑与选区（像素手术）

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Ctrl+Z** | 撤销 | 像素艺术最常用的快捷键之一——试一笔、不对、Ctrl+Z |
| **Ctrl+Y** | 重做 | 撤销的反操作 |
| **Ctrl+A** | 全选 | 选中整个画布 |
| **Ctrl+D** | 取消选择 | 取消当前选区 |
| **Ctrl+X** | 剪切 | 剪切选区到剪贴板 |
| **Ctrl+C** | 复制 | 复制选区 |
| **Ctrl+V** | 粘贴** | 粘贴剪贴板内容 |
| **Shift+H** | 水平翻转 | 左右镜像——检查角色不对称、做 2 向角色时的核心操作 |
| **Shift+V** | 垂直翻转 | 上下镜像——做水面倒影时用 |
| **Ctrl+Shift+I** | 反选 | 选中当前选区以外的所有区域 |

## G.7 必备组合拳（肌肉记忆）

| 操作 | 怎么做 | 什么时候用 |
|------|--------|-----------|
| 快速取色 | 按住 **Alt** 的同时点击画布上的颜色 | 每次换颜色时——不用去色板窗口找，直接在画面上点 |
| 快速切换橡皮擦 | 画的时候按一下**右键** | 临时擦除一个像素，松开右键回到铅笔 |
| 移动画布 | 按住**空格键** + 拖动 | 任何时候——不需要切换到 H 工具 |
| 缩放画布 | **Ctrl + 鼠标滚轮** | 任何时候——比按数字键更顺手 |
| 预览实际尺寸 | 按 **1** 或 **Ctrl+0** | 每画 5-10 分钟缩到 100% 看一眼——玩家看到的不是 800% 放大 |

---

> **学习策略：** 第一天把 G.7 的五个组合拳用熟——取色、橡皮擦、平移、缩放、实际尺寸。这五个操作覆盖了 80% 的像素绘画动作。其余快捷键在你遇到对应章节的练习时自然记住——画 tile 时记住网格开关、做动画时记住 Enter 和 Ctrl+D、做角色包时记住 Shift+H 翻转。
