## 画面问题

| 症状 | 原因 | 解决方向 |
|------|------|---------|
| 角色在引擎里糊了 | Bilinear 过滤没关 | 制作07 第 7.2.1 节 |
| 画面上下滚动时 tile 之间出现 1px 黑线/白线 | 相机坐标是浮点数，tile 网格和屏幕像素不对齐 | 制作07 第 7.3.2 节（相机坐标取整） |
| 角色站着不动但画面有微微颤抖 | 相机紧贴角色做了浮点 lerp() | 制作07 第 7.3.3 节（死区设置） |
| 画面缩放后变模糊 | 非整数倍缩放 | 制作07 第 7.2.2 节 |
| 同一角色左右翻转后"怪怪的" | 光照方向翻转或不对称细节泄露 | 制作02 踩坑六（翻转陷阱） |
| 导出的 PNG 边缘有白边/黑边 | Alpha 通道有半透明像素残留 | 制作07 第 7.3 节（检查 4） |

## 设计问题

| 症状 | 原因 | 解决方向 |
|------|------|---------|
| 所有 UI 元素"一样重要"，没有焦点 | 信息层级没分层 | 制作04 第 4.6 节（布局层次） |
| 按钮看起来都一样，分不清能不能点 | 缺少控件四态设计 | 制作04（UI 控件四态规范） |
| 像素角色画出来翻转后"武器换手了" | 不对称细节镜像泄露 | 制作02 踩坑六（翻转陷阱） |
| 角色在场景里"找不到"，被背景吞了 | 角色明度太接近背景 | 练手04（明度优先表） + 制作03 第 3.1 节 |
| 场景看起来很"挤"，没有呼吸感 | 缺少负空间/呼吸点 | 练手03 第 3.2.4 节 |
| 多个角色放一起看起来"不是一家" | 线宽/色板/细节密度不统一 | 制作09（10 项审计清单） |
| 地面 tile 铺出来像"同一块砖 copy-paste 了几百次" | tile 变体不够多 | 制作03 第 3.2 节 + 练手06 |
